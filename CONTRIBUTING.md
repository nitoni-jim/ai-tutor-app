# Contributing to AI Tutor App

Thank you for your interest in improving the **AI Tutor App** project!

This repository is a research-focused implementation exploring:

- NLP preprocessing  
- Embedding-based semantic search  
- Retrieval-Augmented Generation (RAG)  
- AI-powered educational tooling  

We welcome contributions that improve code organization, experiment quality, documentation, or performance.

---

## üß© How to Contribute

### 1. Fork the repository
Click the **Fork** button on GitHub to create your own copy of the project.

### 2. Clone your fork
```bash
git clone https://github.com/<your-username>/ai-tutor-app.git
cd ai-tutor-app
```

### 3. Create a feature branch
```bash
git checkout -b feature/my-improvement
```

### 4. Make your changes
- Follow clean coding practices  
- Add docstrings and comments  
- Ensure functions are modular and clear  
- Avoid breaking existing structure unless necessary  

### 5. Run tests (coming soon)
```bash
# No tests yet ‚Äî this will be implemented soon
```

### 6. Commit your changes
```bash
git commit -m "Add: brief summary of improvement"
```

### 7. Push your branch
```bash
git push origin feature/my-improvement
```

### 8. Submit a Pull Request
Go to your fork and click **New Pull Request**.  
Explain:

- What you changed  
- Why the change is necessary  
- Any relevant files or modules affected  

Contributions will be reviewed and merged if appropriate.

---

## üìè Code Style Guidelines

To ensure consistency:

- Keep modules small and focused  
- Add docstrings for every function  
- Use type hints where possible  
- Maintain clear separation of concerns  
- Write clean, readable, well-commented Python  
- Keep the research notebook reproducible
