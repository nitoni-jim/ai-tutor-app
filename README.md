# AI-Tutor-App
**Repository (research + engineering style)**  
**Owner:** nitoni-jim

## Abstract
AI-Tutor-App is an adaptive exam-preparation research project focused on WAEC/NECO/JAMB curricula for Nigerian learners. The system integrates NLP preprocessing, embedding-based retrieval, and retrieval-augmented generation (RAG) to provide curriculum-aware question delivery and personalized learning paths.
